Prevádzkový denník

Aplikácia pre továrne a elektrárne ktorá nahrádza papierovú verziu prevadzkových denníkov

View project on GitHub

Hlavné funkcie

  • Vzdialené pripojenie, real-time aktualizácia
  • Používateľské kontá s rôznymi privilégiami
  • Možnosti zobrazenia: Zmena, posledné dva dni, určitý deň, definovaný rozsah a ďalšie
  • Filter udalostí, pokročilé vyhľadávanie
  • Tabuľka s údajmi
  • Ochrana heslom
  • Databáza porúch / opráv
  • Zálohovanie
  • Možnosť tlače a exportu do Microsoft Excel
  • Vzdielaná administrácia a zálohovanie
